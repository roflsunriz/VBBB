# 追加機能リスト

## 追加機能

- 投稿成功後は書き込み欄が自動で閉じるようにする。
- 投稿成功後はスレを差分自動更新する。
- 自分が書き込んだレスをハイライトする機能と書き込んだレスに対する相手の返信をハイライトする機能を付ける。（設定でオン・オフ可能にする）
- 開いているスレッドも記憶して起動時に自動的に開いて復元する。（タブで開いている場合はそのまま復元する）
- カテゴリも複数タブで開いておけるようにする。
- resources/icon.svgをVBBBのアプリアイコンにする。
- ウィンドウのサイズ（通常/最大化も含めて）を記憶して起動時に復元する。
- タイトルバーを"VBBB"から"VBBB - Versatile BBS Browser vX.X.X"に変更する。（実装したはずなのに効果なし）
- リモート検索のdig.2ch.netは常にタイムアウトするので（サーバーダウン？）ff5ch.syoboi.jpを(例：https://ff5ch.syoboi.jp/?q=%E3%83%86%E3%82%B9%E3%83%88)にURLエンコードして検索タブにレンダリングして使う。dig.2ch.net関連のコードは削除する。
- UIからしたらば掲示板とまちBBSのカテゴリ/スレッドを追加する手段がないのでその手段を追加する。通常、URL入力欄があり、そこにスレッドのURLを入力するとカテゴリ/スレッドが自動読込されて追加されるようにする。
