# 追加機能リスト

## VBBBに追加すべき機能の一覧

- お気に入りの欄のアイテムをドラッグ・アンド・ドロップで入れ替え可能にする。
- スレッド一覧のお気に入りマーク（★）をクリックしてお気に入りから削除できるようにする。（現在は登録のみ）
- お気に入りの欄でフォルダと水平線を作成可能にして整理できるようにする。
- 次スレ作成支援機能はボタンが現れる条件を撤廃する。（条件に合致しなくても作成したい場合があるため）
- スレッドタブとスレッドタイトルの間にスレッドタブにあるボタンを移動する。（タブが多くなると狭いため）
- スレッドタブ右クリックのメニューに外部ブラウザでスレッドのURLを開くメニューを追加する。

## 直すべき細かな改善点やバグ修正などの一覧

- 巡回機能をオンにしていてもスレッドビューが自動更新されない。巡回機能をオンにしていてもスレッド一覧が自動更新されない。
