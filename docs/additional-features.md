# 追加機能リスト

## VBBBに追加すべき機能の一覧

- ヘルプの「VBBBについて」のダイアログボックスにURL(https://github.com/roflsunriz/VBBB)を追加する
- メニューバーに「アップデート」のメニューを追加して、ダイアログボックスが開き、現在のバージョンとリリース済み最新版のバージョン、プログレスバー、更新確認ボタン、更新開始ボタン、キャンセルボタンが表示される。更新確認ボタンを押すと、リリース済みの最新版のバージョンが表示されて（初期状態は空欄）、更新開始ボタンを押すと、更新が開始されるようにする。

## 直すべき細かな改善点やバグ修正などの一覧

- ID別書き込み回数一覧又は名前別書き込み回数一覧を解除するとID別書き込み回数又は名前別書き込み回数に入る前のスクロール位置に戻るようにする。
