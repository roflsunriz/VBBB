# 追加機能リスト

## VBBBに追加すべき機能の一覧

- 書き込み欄にチェックボックスを設置し、書き込み後に自動的に閉じるかそのままにするか選べるようにする。
- ff5ch.syoboi.jpに簡易的な広告ブロック機能を付けてデフォルトでオンにする。ユーザーが任意でブロックルールを編集できるようにする。ブロックルールの書き方を解説したヘルプページを用意する。
- ff5ch.syoboi.jpでスレッドを開いたとき、webViewで開かないようにブロックする。
- ff5ch.syoboi.jpでスレッドを開いたとき、スレッドタイトルが正しく取得できない問題に対処する。
- imgurのアルバムリンク(/a/)でも画像プレビューできるようにする。
- 多数の有名サイトの画像リンクと動画リンクもインライン展開する。インライン展開は設定でオン・オフできるようにする。
- URLは自動認識してクリックして外部ブラウザで開けるリンクに変換する。
- スレッドタブ右クリックで巡回に追加できるようにする。
- スレ一覧タブ右クリックで巡回に追加できるようにする。
- スレ一覧に「勢い」と「人気度（パーセンタイル）」と「完走率(1000レスに対するパーセンテージ)」、「>>1の日付時刻」と「最新レスの日付時刻」を表示する。
- スレ一覧を各項目で並び替えできるようにする。
- スレッドビュー内に画像一覧、動画一覧、リンク一覧、人気レス一覧、コテハン一覧、必死レス（長文）、必死レス（ID回数）、必死レス（ワッチョイ別）のリンクを表示する。各リンクをクリックすると分析結果や一覧が表示される。
- スレ一覧は最後の一個のタブでも閉じられるようにする。
- スレッドビュー内の各レスを右クリックで、名前、本文、URL、名前+本文+URL、本文+URLをコピーできるようにする。
- 画像のプレビューはキーボードの←と→でスレッド内の前の画像と次の画像を切り替えられるようにする。
- 外部板のスレッドを追加したときは、外部カテゴリも板一覧に追加されるようにする。外部カテゴリは任意で削除可能にする。
- 認証、プロキシ、巡回、Cookie/UA、コンソールの各モーダルをリサイズしようとクリックするとモーダルが自動的に閉じてしまう問題に対処する。
- スレッドビュー内の日付時刻は現在時刻から何分前の投稿なのか差分も表示する。設定でオン・オフ可能にする。
- スレ一覧にお気に入りスレがあるとタブ切り替えでお気に入りスレ他のスレ一覧に混ざり同じお気に入りスレが増殖する問題に対処する。
- スレ一覧もお気に入りに追加できるようにする。
- お気に入りタブからも巡回に追加できるようにする。
- 通常の書き込み欄とは別に、プログラマティック書き込み欄を追加する。プログラマティック書き込み欄では指定日付時刻何秒に書き込み、指定秒数間隔で書き込み、指定秒数カウントダウン後書き込み、指定回数同じ名前欄とmail欄と本文で書き込み、または複数の内容の違う指定名前欄+指定mail欄+指定本文で自動書き込みできるようにする。それぞれの条件は自由にオン・オフできるようにする。
- 再読み込みするとスレ一覧がリセットされてしまう（タブが自動的に閉じてしまう）問題に対処する。また、スレッドのスクロール位置自動復帰も動作しない問題に対処する。
- 名前欄のワッチョイにIPアドレスが露出しているスレッドやID部分に代わりにIPアドレスが露出しているスレッドでは、IPアドレスをクリックするとIPアドレスのWhoIsやジオロケーションが表示されるようにする。
- 名前欄にワッチョイがあるスレッドではワッチョイをクリックすると公開情報から回線種別とユーザーエージェントを推定して表示する。
- 書き込み欄に「トリップ」に関するヘルプボタンを設置する。
- スレッド内で書き込み回数の多いIDやワッチョイやコテハンの回数を横に付与する。また、書き込み回数の多いIDやワッチョイやコテハンをクリックするとそのIDやワッチョイやコテハンの書き込み一覧が表示される。
- 板一覧、お気に入り、検索のタブに履歴を追加して閉じたスレッドや閉じたカテゴリ一覧を表示し、開き直せるようにする。履歴データはデータベースに自動保存する。履歴データのマネジメント機能も付ける。
- 板一覧からカテゴリを検索できるようにする。
- 書き込み欄のどんぐりボタンを押すとどんぐりのログイン状況やステータスが表示されるようにする。
- IPアドレスが表示されるスレッド（ID or ワッチョイ欄またはその両方）では自動的に書き込み欄内でプライバシーに関する警告を表示する。

## 上記機能の実装後の直すべき細かな改善点やバグ修正などの一覧

- したらば掲示板でスレッド一覧のタブが「study/13463」のような表示になってしまう。
- したらば掲示板でスレッド名が「study/13463 - 1770190138」のような表示になってしまう。
- まちBBSでスレッド一覧のタブが「tokyo」のような表示になってしまう。
- まちBBSでスレッド一覧の中身が空になっている。
- まちBBSでスレッド名が「tokyo - 1182428917」のような表示になってしまう。
- まちBBSのスレッドが正しくレンダリングできていない。(837 852 [東京都名無区民] sage 2025/08/09(土) 11:00:23 ID:hVU73KcA)の表示になっていて、特に名前欄と本文が取得できていない。本文が表示されるべき箇所に日付とIDが表示されてしまう。名前欄が「名前： 東京都名無区民 」になるべきところが数字になっている。
- お気に入りタブでスレッドをクリックしてもスレッドを開くことが出来ない。
- スレッドビューで最初または最後のレスまで来たときに更にスクロールしようとすると（上又は下にスクロールする場合）アイコンが表示されてレッドが手動更新されてほしい。スレッドビューにも手動更新ボタンを付ける。
- リモート検索(ff5ch.syoboi.jp)で検索してスレッドを開くと確定でスレッド名が「study/13463 - 1770190138」や「tokyo - 1182428917」のような表示になってしまう。（5chのスレッドは板一覧から辿ると正常だが、リモート検索から辿るとスレッド名が変になる）。
- リモート検索からスレッドを開くと自動でスレッド一覧を表示してくれない不具合。
- スレッドビューに手動更新ボタンを付ける。
- 書き込み欄、プログラマティック書き込み欄、分析結果欄、NG管理欄はリサイズ可能にしたい。
- したらば掲示板に書き込もうとするとgrtError。403エラー。SlevoとgikoNaviG2ソースコードにヒントがあるかもしれないので参照して修正する。まちBBSに書き込もうとすると404エラー。
- ローカル検索の検索結果が文字化けしている。
- スレッド一覧も端っこ（上又は下にスクロールする場合）でスクロールすると手動更新されるようにしたい。
- 書き込み欄のどんぐりボタンクリックで表示されるべきは、ログイン済みのどんぐりアカウントの警備員のレベルや大砲などのどんぐりシステム特有のステータスです。どんぐりアカウントがログインしていない場合はログインフォームが表示されるようにしたい。
- スレッド一覧とスレッドのタブの右クリックメニューに「更新」を追加する。
