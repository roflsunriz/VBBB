# 追加機能リスト

## VBBBに追加すべき機能の一覧

- プログラマティック書き込み欄を拡張して、独自DSLで書き込みを実行できるようにする。独自DSLでは、名前、mail、本文、各書き込み間のカウントダウンや間隔や繰り返し、指定日時などを指定できるようにする。プログラマティック書き込み欄でそのプログラムファイルを読み込み、実行ボタンを押すと書き込み開始する仕様にする。

## 直すべき細かな改善点やバグ修正などの一覧

- したらば掲示板でIDがレンダリングされない（ID別書き込み回数のバッジは表示されている）
- スレッドがDAT落ちになるとスレッド更新後スレッド表示が空（0レス）になる不具合
